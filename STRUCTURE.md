# 📁 هيكل المشروع المحدث - معهد بنين المنصورة النموذجي

## نظرة عامة
تم تعديل الهيكل ليكون أكثر بساطة مع إبقاء ملفات HTML في المجلد الرئيسي لضمان عمل الروابط والصفحات بشكل مباشر، مع تنظيم ملفات التنسيق والبرمجة في مجلدات خاصة.

## 🗂️ البنية الحالية

```
azhar1/
├── 📄 index.html                 # الصفحة الرئيسية
├── 📄 about.html                 # عن المعهد
├── 📄 admin.html                 # لوحة التحكم
├── 📄 downloads.html             # صفحة التحميلات
├── 📄 news.html                  # صفحة الأخبار
├── 📄 sheikh.html                # كلمة شيخ المعهد
├── 📄 stages.html                # المراحل الدراسية
├── 📄 README.md                  # ملف التوثيق الأصلي
├── 📄 STRUCTURE.md               # هذا الملف (شرح الهيكل)
│
├── 📁 css/
│   └── style.css                 # ملف التنسيق
│
├── 📁 js/
│   ├── app.js                    # السكريبت الرئيسي
│   ├── admin.js                  # سكريبت لوحة التحكم
│   └── supabase.js               # تكامل Supabase
│
└── 📁 assets/
    ├── 📁 images/                # الصور
    └── 📁 data/                  # البيانات
```

## ✅ التعديلات المنجزة
1. **ملفات HTML**: تم إعادتها جميعاً إلى المجلد الرئيسي (Root).
2. **ملفات CSS/JS**: بقيت منظمة داخل مجلدات `css/` و `js/`.
3. **تحديث الروابط**: تم تعديل جميع المسارات لتكون مباشرة (مثال: `href="css/style.css"`).
